
-- SUMMARY --

Supports adding active theme files to page via context


-- REQUIREMENTS --

* Context-3.x - http://drupal.org/project/context
* Chaos Tool Suite - http://drupal.org/project/ctools

-- INSTALLATION --

* Enable module in admin/build/modules


-- CONFIGURATION --

* The module adds a "Add JS" reaction in the Context UI.
* The module adds a "Add CSS" reaction in Context UI


-- ROADMAP --

* Add sorting support
